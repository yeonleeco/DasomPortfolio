This is my very first User Interface Design called, "Plus Design".
This is a virtual site selling the design software program like Adobe CC programs
and people can purchase by choosing the plans.

The "Features" page contains several pictograms that implicitly show the features of the software program. With a carousel layout, users can navigate through each feature by pressing the arrows on either side. Along with the topic of "Design Software Program", it shows more diverse design aspects as these pictograms.

A wide layout is used in "Pricing" page because it's important to show to users
what are the differences for each plan. By using the same color in logo for each
plan box, it gives the unity of a whole page.
